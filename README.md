<h1> Project AdsDetect</h1>
<h2> Requirements </h2>
<p> 1. python</p>
<p> 2. npm </p>
<h2> Installation and running process</h2>
<p> Go to the project folder</p>
<p> run ./install.sh</p>
<p> pip3 install scikit-learn</p>
<p> pip3 install Levenshtein</p>
<p> cd api </p>
<p> npm run prod</p>
<p> Application is available at http://localhost:3000</p>
